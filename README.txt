This is my attempt at making a boolean equation converter program. 
Converts simple Boolean Equations into their Nand/Nor only forms.
Right now only supports the following operators:
• NOT, !, ¬
• AND, *
• OR, +
• NAND, !*
• NOR, !+
• Parenthesis are supported - ()

How to use:
1. Run program
2. provide a valid boolean expression
   2.1 Each element MUST be seperated by spaces.
   2.2 Example: Valid - A OR B / X + J
3. Enjoy

This is version 1 so it's pretty barebones.
Programing will *probably* be updated in the future to support more operations, symbols and better UI.
